# PI-KAN 训练结果与 KHAOS 指标映射说明

本文档说明 iter9 训练后的 breakout / reversion 双核语义如何映射到 `KHAOS.pine` 的当前输出。KHAOS 已经彻底废弃了复杂的数据面板和多线条堆叠，采用**单轴动能基准 + 颜色映射状态**的极简直觉设计。

## 1. 当前对外语义

| 指标视觉元素 | 当前含义 | 读取方式 |
| :--- | :--- | :--- |
| **背景色 (Background)** | 当前主状态属于哪一类环境 | 橙色：Breakout 环境（无方向预警）；蓝/紫：Reversion 环境（带方向）；灰色：Neutral/Filter 观望 |
| **动能主线 (Momentum Baseline)** | 市场的多空惯性与状态映射 | 线在零轴上方为多，下方为空。线的颜色代表当前被激活的物理状态核。 |
| **系统能量柱 (System Energy)** | 底层物理能量（波动率+MLE+熵）的综合强度 | 柱子收缩代表蓄势，放大代表释放。方向跟随主线多空。 |

## 2. 双核与方向的映射关系

### 2.1 Breakout 核 (无方向的能量释放)

**核心修正：** 
在 PI-KAN 的训练中，Target 0（突破核）的本质是预测“波动率扩张” (Volatility Expansion)。物理学上的能量激增本身是**标量（无方向的）**。
当前的 Pine 脚本已经**剥离了突破核的多空方向判定**，恢复了其本质：

- `Breakout (橙色)`：代表市场从压缩进入释放，波动率开始扩张。这是一种**无方向的动能预警**。交易者看到橙色背景时，应结合主图K线结构或动能线位置，顺势而为，而不是依赖系统给出的死板多空方向。
- `Neutral/Filter (灰色)`：说明突破条件未满足，或被回归型极端失衡压制。

### 2.2 Reversion 核

reversion 核来自一组“极端失衡后回归”的代理量判断，重点看：

- EKF 残差与 EMA 偏离是否同步达到极值；
- 方向是否已经明确落在 bull 或 bear；
- breakout 的延续压力是否仍然过强，导致回归条件尚不可靠。

Pine 中的 reversion 输出强调的是“确认型回归”，不是任意偏离都触发。只有当方向确认足够清晰时，才会进入 bull reversion 或 bear reversion；否则仍然回到 neutral/filter。

## 3. neutral/filter 为什么是显式状态

iter9 之后，neutral/filter 不再被视为“没有信号的空白期”，而是一个必须被诚实表达的状态：

- breakout 与 reversion 都未达到有效触发条件；
- 虽然出现了异动，但 bull / bear 的方向结算仍不够清晰；
- 两个核都在争夺主导权，当前更适合等待而不是强行归类。

这意味着 KHAOS 当前的零值与灰色过滤背景，本身就是有效输出，含义是“当前阶段更适合过滤，而不是勉强给出方向”。

## 4. breakout 的有限周期感知表达

本轮脚本遵守“全周期统一逻辑”，没有为不同周期拆成多套判定规则，但对 breakout 做了有限的周期感知表达：

- 5m / 15m：breakout 维持正常表达；
- 1h：breakout 视觉表达略收敛；
- 4h：breakout 明确按“高周期稀疏确认核”展示；
- 1d 及更高：breakout 仅保留稀疏确认定位，不制造与低周期相同的触发密度假象。

这里的周期感知只作用在输出映射与阅读提示层，不改变 breakout / reversion 的统一底层逻辑，也不会人为放大高周期 breakout 强度。

## 5. 当前图表应该怎么读

新的读图逻辑将变得极其简单且严谨：

1. **先看背景色**：判断大环境。平时只看灰线在零轴上下游走。
2. **一旦背景变橙**：立刻警觉，要有大行情了（突破）。但不预判方向，而是结合主图看突破哪边。
3. **一旦背景变蓝/紫**：立刻准备逆势操作，偏离过头了，准备做反转（蓝多，紫空）。
4. **看能量柱起伏**：判断这是刚刚发动的信号，还是已经是强弩之末。

## 6. 与旧版叙事的区别

当前版本已经不再采用以下旧口径：

- 不再强制给突破 (Breakout) 附加多空方向；
- 不再使用容易引起歧义的圆点作为切换预警；
- 不再使用 Force 作为 Pine 主表达；
- 不再使用“红蓝粗线 = 固定阈值危险区”的单层解释；
- 不再把 neutral/filter 简化成“图上没东西所以忽略”；
- 不再默认 breakout 在所有周期具备相同密度。

新的主叙事是：**先判断当前属于 breakout、reversion 还是 neutral/filter，再判断方向 bull / bear，最后结合 breakout lens 理解周期适配差异。**
