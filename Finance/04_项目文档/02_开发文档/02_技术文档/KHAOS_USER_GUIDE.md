# KHAOS PI-KAN 指标使用与判读指南

本文档将帮助您快速理解 KHAOS_INTERP (Interpretable) 指标的各项功能、视觉含义及实战用法。

## 1. 核心视觉元素 (Visual Elements)

### 1.1 主图线条 (Force Line)
这是指标的核心，代表物理系统中的**“瞬时回归力 (Restoring Force)”**。
- **灰色细线**: 市场处于平衡态，无显著物理驱动力。
- **蓝色粗线**: **空头预警 (Bear Warning)**。
    - 含义：Force > 阈值。价格偏离过高，物理力场正在将其“拉回”。
    - 操作：警惕回调，考虑做空。
- **红色粗线**: **多头预警 (Bull Warning)**。
    - 含义：Force < -阈值。价格超跌，物理力场正在将其“推回”。
    - 操作：警惕反弹，考虑做多。
- **透明度变化**: 线条越实（不透明），代表信号置信度越高；线条越虚（透明），代表置信度较低。

### 1.2 动态填充 (Dynamic Gradient Fill)
- **视觉效果**: 零轴与 Force 线之间的区域会根据信号强度进行填充。
- **颜色含义**: 
    - **蓝色填充**: 下行压力 (Bearish Pressure)。
    - **红色填充**: 上行压力 (Bullish Pressure)。
- **深浅含义**: 颜色越深，代表信号置信度 (Confidence) 越高；颜色越浅或变灰，代表物理状态无效 (Chaos/Low Vol)。这种设计比单纯的背景色带更干净、更专业。

### 1.3 交互式标签 (Interactive Labels)
- **功能**: 指标会自动在 Force 信号显著的 K 线上方显示一个悬浮标签。
- **内容**: 包含该时刻的 `Force`, `Hurst`, `Entropy`, `Conf` 四大核心数据。
- **用途**: 替代传统的数据窗口 (Data Window)，让您无需移动视线，直接在图表上看到关键时刻的物理状态。

## 2. 灰色主线与 EKF 的关系 (Gray Line & EKF)

### 2.1 什么是灰色主线？
灰色主线代表 **Force (回归力)** 处于 **物理平衡态 (Equilibrium)**。
- **定义**: 当 Force 的绝对值小于阈值 (1.65) 时，系统认为当前价格运动属于 **随机游走 (Random Walk)** 或 **布朗运动**。
- **物理意义**: 此时市场中没有显著的“势能”积累，价格就像一个自由漂浮的粒子，没有被弹簧拉回或推开。

### 2.2 与 EKF (扩展卡尔曼滤波) 的关系
PI-KAN 模型内部集成了一个 EKF 状态估计器，它实时计算价格的“真实位置” ($x$) 和“速度” ($v$)。
- **灰色主线 = EKF 预测准确**: 当灰色线接近 0 轴时，意味着当前价格走势完全符合 EKF 的惯性预测。市场处于“合乎逻辑”的运动中，没有发生意外的相变。
- **红蓝爆发 = EKF 预测失效**: 当 Force 突然爆发（变红/变蓝）时，意味着价格行为偏离了 EKF 的物理预测（Innovation 变大）。这种偏离就是**交易机会**——市场正在通过剧烈的反向运动来修正这个偏离。

## 3. 数据面板详解 (Data Panel)

位于图表右上角的面板提供了实时的物理状态诊断。

| 字段 | 显示内容 | 物理含义与判读 |
| :--- | :--- | :--- |
| **物理状态 (Regime)** | **混沌 (Chaos)** | 灰色。熵值过高，市场随机游走。**休息。** |
| | **低波 (Low Vol)** | 蓝色。波动率过低，能量不足。**休息。** |
| | **强趋 (Trend)** | 橙色。Hurst 指数极高，趋势惯性大，不易反转。**顺势或观望。** |
| | **有效 (Valid)** | 绿色。物理结构清晰，适合捕捉反转。**关注信号。** |
| **Hurst** | 0.0 ~ 1.0 | 分形维。数值越低，均值回归越强（适合震荡）；数值越高，趋势越强。 |
| **Entropy** | 0.0 ~ 1.0 | 熵/复杂度。数值越低，市场越有序；数值越高 (>0.9)，市场越混乱。 |
| **Vol (Z)** | -3.0 ~ 3.0 | 波动率的 Z-Score。负数代表波动低于平均水平。 |
| **力场强度 (Force)** | -2.0 ~ 2.0 | PI-KAN 模型的直接输出。绝对值越大，反转动力越强。 |
| **置信度 (Conf)** | 0% ~ 100% | 基于 Force 强度与 SES 预测的综合评分。>80% 为强信号。 |

## 3. 如何判断信号节点 (Signal Logic)

结合物理参量，判断信号的有效性分为三个层次：

### 3.1 观察物理状态 (L1: Regime)
首先看右上角面板的 **物理状态 (Regime)**：
- **Chaos (灰色)**: 市场混乱，Entropy > 0.9。此时 Force 信号不可信，**观望**。
- **Low Vol (蓝色)**: 波动率过低，能量不足。此时信号容易失效，**观望**。
- **Valid (绿色)**: 物理结构健康，适合交易。**进入下一步**。

### 3.2 确认力场方向 (L2: Force)
观察主图 Force 线条：
- **蓝色粗线**: Force > 1.65。下行恢复力极强，预期价格下跌。
- **红色粗线**: Force < -1.65。上行恢复力极强，预期价格上涨。
- **灰色细线**: 市场处于平衡态，无方向性驱动力。

### 3.3 验证置信度 (L3: Confidence)
观察线条的**实虚程度**和面板上的 **置信度 (Conf)**：
- **实线 (Conf > 80%)**: 强信号，大概率反转。
- **虚线 (Conf 50-80%)**: 弱信号，需配合其他技术分析（如支撑阻力位）确认。
- **透明 (Conf < 50%)**: 信号微弱，忽略。

## 4. 特别说明：高波动模式 (High Volatility Mode)

**v4.0 更新提示**:
- KHAOS 现在默认使用 **v4.0 (High-Vol)** 模型，专注于加密货币、黄金和原油等高波动资产。
- **响应速度更快**: 相比 v3.0，新模型对 5m 周期的反转反应更敏锐，滞后性大幅降低。
- **风险提示**: 更快的信号意味着更高的噪声风险。在低波动市场（如横盘的外汇）中，可能会出现假信号。
- **面板标识**: 状态面板现在会显示 `(Live)` 或 `(Hist)`，提示您当前数据是实时计算还是历史回测。

## 5. 常见问题 (FAQ)

### Q: 为什么有时候看到了红线，但面板显示“混沌”？
**A**: 这是因为 Force 虽然很大，但 Entropy 超标了。这时候红线会变灰或消失（取决于设置），或者背景变成灰色。这提示您：**虽然看起来像机会，但市场太乱，胜率不高，建议放弃。**

### Q: 怎么查看历史某时刻的数据？
**A**:
1.  **数据窗口 (Data Window)**: TradingView 右侧边栏的“数据窗口”图标（Data Window）。
2.  **操作**: 将鼠标移动到 K 线图上的任意时间点。
3.  **查看**: 数据窗口会显示该时刻的 `Hurst`, `Entropy`, `Volatility Z`, `Confidence` 等数值。
    *注：图表上的面板 (Table) 由于技术限制，通常只显示最新一根 K 线的数据。回溯历史请使用右侧数据窗口。*

### Q: 灰色主线是什么意思？
**A**: 灰色主线表示 **Force 在阈值范围内 (-1.65 ~ 1.65)**。这意味着价格处于“物理平衡态”，没有受到强烈的回归力或推动力，市场可能正在进行正常的布朗运动。此时不应进行左侧抄底或摸顶。
