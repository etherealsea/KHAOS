## 2. 架构优势验证（对用户问题的响应）

本次提取结果充分印证了当前架构对 KAN（柯尔莫哥洛夫-阿诺德网络）和注意力残差（Attention Residual）机制的深度利用：

### 2.1 KAN 架构的白盒优势
KAN 的核心优势是**高维非线性拟合能力**和**白盒可解释性**（通过样条函数 Splines 替代线性权重）。
通过对新训练的模型进行符号提取，我们不仅看到了它的拟合结果，更**逆向提取出了明确的物理临界点阈值**，这证明模型并非黑盒，而是真正学到了分段非线性的物理规则：
- **混沌突破相变（基于 Target 0 波动率扩张）**：
  - `Volatility > -0.49` 时启动激增模式（放大 1.1 倍）。
  - `Entropy > 0.54` 时启动激增模式。
  - `MLE > 0.17` 时启动激增模式。
- **极值反转相变（基于 Target 1 均值回归）**：
  - `abs(EKF_Res) > 1.00` 或 `abs(EMA_Div) > 1.00` 时触发极强的均值回归惩罚。
- **环境基准**：
  - `Hurst > -0.90` 时系统处于趋势态（相较于死水期更容易放大波动）。

### 2.2 注意力残差机制 (Attention Residual) 的时序优势
纯 KAN 架构本质是前馈的，如果直接输入时序序列会丢失时间因果关联。Kimi 提出的 Attention Residual Block 完美弥补了这一缺陷。从最新的注意力分布提取结果中可以看出：
- **远期记忆主导**：模型对 `t-8 之前` 的远期 K 线分配了高达 **59.3%** 的注意力权重，证明它高度依赖长周期的上下文（如大级别的阻力支撑或长周期物理特征累积）。
- **局部触发锚点**：模型在 `t-3` 位置分配了单点最高的注意力权重（**8.4%**）。这意味着模型在吸收了长周期背景后，会将近期的某一根异动 K 线（如 `t-3` 时的巨量残差脉冲）作为确认相变的**核心触发器 (Trigger Anchor)**。

## 4. Pine Script 落地与可视化重构总结 (2026-03-24 补充)
在完成模型阈值的映射后，我们发现旧版 Pine Script 的可视化（图表中的杂乱曲线与柱状图）可读性极差，且缺乏明确的交易指导意义。针对此问题进行了彻底重构：
- **剥离杂音**：移除了将动能速度（EKF Velocity）、残差冲击和混沌指数强行捏合在一个副图里的做法（量纲不一致导致图表被过度压缩）。
- **全局状态感知 (Background)**：使用直观的背景色（如红/绿代表突破，蓝/黄代表反转）来标示当前所处的“相变环境”。死水期则为无色。
- **降维打击**：主线只保留一根灰色的“波动率扩张率”基准线，这正是神经网络 `Target 0` 的代理。
- **明确交易信号 (Labels)**：在相变状态刚发生切换的那一根 K 线上，直接打上明确的文本标签（如 `B-OUT` 牛市突破, `REV-UP` 触底反转），做到看图即交易，彻底消除了视觉歧义。
