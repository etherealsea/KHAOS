## 1. 项目概况与核心设计
本项目旨在解决传统深度学习在金融时序预测中存在的**过拟合**与**分布偏移 (Distribution Shift)** 问题。我们不再追求单一的“价格预测”，而是构建一个**物理信息增强 (Physics-Informed)** 的神经符号系统，专注于探测市场的**相变 (Phase Transition)**——即从“有序趋势”到“混沌震荡”的机制切换。

### 1.1 核心技术架构 (The PI-KAN Framework)
我们提出了 **PI-KAN** 五维架构，将物理先验与神经网络深度融合：
*   **P (Physics-Informed)**: 在损失函数中引入物理约束（如动能守恒约束、熵增抑制约束），强迫 AI 遵守市场物理规律。
*   **I (Inference via EKF)**: 利用 **扩展卡尔曼滤波 (EKF)** 作为“物理传感器”，实时分离价格的**势能 (Trend)** 与 **动能 (Kinetic Energy)**。
*   **K (KAN Architecture)**: 采用 **Kolmogorov-Arnold Network**，利用其样条激活函数的特性，实现对非线性残差的高精度拟合与可解释性。
*   **A (Anomaly Detection)**: 结合 **Hurst 指数** 与 **排列熵 (PE)**，量化市场的长记忆性与混沌度，识别趋势衰竭的拐点。
*   **N (Normalization)**: 独创“系统内生归一化”，利用 EKF 内部的**创新协方差 ($S$)** 动态调整指标灵敏度，彻底解决参数迟滞问题。

## 3. 当前遇到的挑战与解决方案 (Key Challenges)

### 3.1 挑战：ES 期货的“背景色溢出”
*   **问题**: ES 等具有非交易时段截断的资产，在开盘瞬间会产生巨大的价格跳空 (Gap)，导致 EKF 预测误差瞬间飙升至 $10^{5}$ 量级，破坏图表渲染。
*   **解决**: 
    *   弃用单纯的线性缩放。
    *   引入 **Hard Clamping** 机制，在 EKF 输出端强制将 Z-Score 限制在 $[-10, 10]$ 的物理有效区间内，确保视觉稳定性。

### 3.2 挑战：Pine Script 动态周期限制
*   **问题**: Pine Script v6 禁止 `ta.stdev(source, length)` 中的 `length` 为变量，导致无法直接实现“随 Hurst 指数变化的动态窗口”。
*   **解决**:
    *   **数学等价替换**：利用指数移动平均 (EMA) 的平滑因子 $\alpha$ 与窗口长度 $N$ 的关系 $\alpha = \frac{2}{N+1}$。
    *   动态计算 $\alpha$ 代入 EMA 公式，从而在不违反语法规则的前提下，实现了真正的动态周期波动率计算。
