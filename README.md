# 基于物理信息增强神经网络（PI-KAN）的金融时间序列相变探测研究

这个仓库用于同步项目的代码、文档、实验脚本与 AI 协作资料，方便在网页端 AI 或其他设备上继续推进工作。

## 目录概览

- `Finance/02_核心代码/`: 核心实现、工具脚本、历史代码归档
- `Finance/03_实验与验证/`: 实验脚本、验证流程与阶段性结果
- `Finance/04_项目文档/`: 规划、论文材料、技术文档、实验报告
- `.trae/` 和 `Finance/.trae/`: AI 协作规则、技能和计划文档

## 当前同步策略

为了让远程仓库更适合网页端 AI 读取，并避免大文件上传限制，以下内容默认不纳入 Git 版本控制：

- 原始行情数据和研究处理数据
- 模型权重、pickle 缓存和实验结果产物
- 运行日志、`__pycache__` 和其他本地临时文件

如果后续确实需要把大文件一并同步，建议再补充 Git LFS 或单独的数据存储方案。
