# 统一局部物理周期并重构均值回归目标 Spec

## 为什么 (Why)
当前方案虽然已经把训练目标拆成了“波动率扩张”和“均值回归”两个连续目标，但均值回归目标仍然主要由价格相对 EMA 的偏离与未来收益构造，导致训练监督过于价格中心化，未能充分让 PI-KAN 在统一周期下自主判断 EKF、Hurst、MLE、Entropy 等物理信息对“即将回调/反转”的真实贡献。与此同时，Pine 与训练侧的局部窗口长度仍然混用 10、14、20 等多套周期，削弱了体系一致性与解释性。

## 做了哪些更改 (What Changes)
- **BREAKING**: 统一局部物理量的主周期基准为 `20`，包括训练侧与 Pine 侧的局部物理特征窗口、标准化窗口以及局部背景感知窗口。仅允许 2~3 的短平滑作为显示或数值稳定用途。
- **BREAKING**: 重构均值回归目标，不再将其定义为单纯的 `EMA 偏离 × 未来收益` 交叉项，而是升级为“即将发生有效回调/反转的连续强度目标”。
- 均值回归目标与输入结构必须保留完整的 KHAOS 物理候选信息，包括但不限于：
  - EKF 公允轨道偏离 / EKF 残差
  - EMA20 偏离
  - 局部波动率状态
  - Hurst 趋势背景
  - MLE / Entropy 混沌背景
  但系统**不得预先武断规定**这些信息在回归核中必须等权或必须全部直接进入单条显式规则，而应让 PI-KAN 在统一周期下自行学习哪些组合真正对爆发和回归有效。
- 训练与推理逻辑必须区分：
  - **爆发核**：用于识别系统能量释放与波动率扩张
  - **回归核**：用于识别价格已进入“即将迎来可交易级别回调/反转”的失衡区
- Pine 侧的超买/超卖判定必须与新的回归目标语义对齐，不能仅由简单阈值拼接决定。

## 新增需求 (ADDED Requirements)
### 需求：统一局部物理周期
系统必须以 `20` 为局部物理信息的主观察窗口，保证训练侧与 Pine 侧的局部统计、背景判断和标准化基准具有一致的时间尺度。

#### 场景：统一局部窗口
- **当** 系统计算 Hurst、波动率背景、局部熵、MLE 背景、残差标准化和均值偏离背景时
- **则** 应优先使用围绕 `20` 的统一主窗口，而不是混用多个彼此独立的中短周期

### 需求：物理增强的均值回归目标
系统必须提供一个新的连续回归目标，用于刻画“当前时点是否已进入即将发生一定程度回调甚至大反转的区间”。该目标本身不能再只由价格偏离项构造；同时，训练输入必须保留完整 KHAOS 物理状态，交由 PI-KAN 在统一周期下自行学习哪些信息对回归最有效。

#### 场景：有效回调 / 反转样本
- **当** 当前价格明显偏离 EKF 公允轨道与 EMA20，且后续窗口内出现了足够显著的反向价格推进
- **则** 新的均值回归目标应给出显著正值，并按回调/反转强度连续放大

#### 场景：伪极值样本
- **当** 当前价格虽然偏离均值中枢，但后续仅出现微弱回踩或继续顺势扩张
- **则** 新的均值回归目标不得将其视为强回归样本

### 需求：回归核保留完整物理背景
系统必须允许 PI-KAN 在均值回归子任务中继续访问 Hurst、MLE、Entropy、Volatility、EKF、EMA20 偏差等完整物理背景，以便由网络自主学习哪些组合更容易产生真正的回调与大反转；系统不得在规则层先验地把这些信息硬性裁剪为少数固定条件。

#### 场景：网络学习复合条件
- **当** 模型接收训练窗口
- **则** 均值回归输出应由完整物理状态共同驱动，而不是提前在目标设计阶段舍弃关键物理背景

## 修改的需求 (MODIFIED Requirements)
### 需求：多任务目标定义
系统必须继续保留双输出结构，但第二输出不再被定义为简单的“均值偏离 × 未来收益”的回归代理，而应被定义为“物理增强的即将回调/反转强度”。该输出的监督设计应鼓励模型利用完整物理候选信息，而不是在目标定义阶段先行舍弃非价格特征。

### 需求：Pine 中的超买超卖区间
Pine 侧的超买/超卖区间必须映射自新的回归核语义，优先表达“即将进入有效回调/反转窗口”，而不是单纯表达“已经偏离过大”。

## 移除的需求 (REMOVED Requirements)
### 需求：均值回归目标仅由价格偏离与未来收益构成
**原因**：该定义仍然过于价格中心化，未充分给 PI-KAN 提供在统一周期下自主学习完整物理结构的空间，容易把普通偏离与真正的可交易级回调/反转混为一谈。
**迁移**：升级为“以更准确的回调/反转结果为监督，并保留 EKF、EMA20 偏差、波动背景、混沌背景等完整物理候选信息供网络学习”的连续目标。
