### 步骤二：深度剖析信号失效原因 (Failure Analysis)
针对“信号准确度极低”这一新发现的问题，总结可能导致 Python 训练结果与 Pine 实盘指标脱节的根本原因（不修改代码，仅做理论探讨），例如：
* **计算机制差异**：Python 中对 MLE、Hurst、Entropy 的全量/批量计算，与 Pine Script 中基于 `ta.ema` 和固定回溯窗口的流式计算存在数学表达上的偏差。
* **绝对阈值的局限性**：决策树提取的绝对阈值（如 0.27, 0.70）可能在不同的时间周期（如 1h vs 5m）或不同的资产波动率基准下并不通用。
* **潜在的前视偏差 (Lookahead Bias)**：在 Python 特征工程中计算 Hurst 或进行 Z-score 归一化时，可能不经意间引入了未来数据，导致训练集表现好但实盘（指标）表现差。
